**In the project directory, you can run:**

### `npm install i`
To install npm
### `npm run dev`
To start the app in dev mode.\
Run the front-end part then open [http://localhost:3000](http://localhost:3000) to view it in the browser.
### `npm start`
For production mode
### `npm run test`
Run the test cases.
